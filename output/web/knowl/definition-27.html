Definition 4.15.1. Momentum.

The momentum of a single object is defined as the mass of the object times its velocity: \(\vec{p} = m\vec{v}\text{.}\) For a system with multiple objects, the momenta of each object in the system can be summed: \(\vec{p}_{system} = \Sigma \vec{p}_i = \vec{p}_1 + \vec{p}_2 + \vec{p}_3 + \dots\text{.}\)
